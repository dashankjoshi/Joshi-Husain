# Raw Data

This folder is for the raw datasets used in the project.

Files:
- `weather_2019.csv`: NOAA Global Hourly weather dataset.
- `flights_2019.csv`: BTS Airline On-Time Performance dataset.

## Flight Dataset

The BTS flight dataset (`flights_2019.csv`) is not stored directly in GitHub because the file is larger than GitHub's web upload limit and exceeds 50MB.

Download link: [BTS Flights 2019 Dataset](https://uofi.box.com/s/y5htwy3v97jklrw3xscm00d57okd264p)
